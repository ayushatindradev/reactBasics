## 📜 Installation Guide

To run this project on your local machine:

```bash
# Clone the repository
git clone https://github.com/yourusername/react-tutorial.git

# Navigate into the project
cd react-tutorial

# Install dependencies
npm install

# Start the development server
npm run dev
```
